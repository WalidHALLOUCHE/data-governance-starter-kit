## 🔧 Modèle de données

5 tables principales :
1. **donnees_finance_powerbi** — Données financières nettoyées
2. **rapport_anomalies_powerbi** — Détail des anomalies détectées
3. **synthese_qualite_powerbi_ready** — Score qualité par source
4. **score_qualite_global_powerbi** — Indicateurs de validité globale
5. **statistiques_anomalies_powerbi** — Statistiques par type d'anomalie

## 📋 Pages du Dashboard

1. **Executive Summary** — KPIs clés, CA/EBITDA, anomalies par gravité
2. **Qualité & Gouvernance** — Taux validité, score qualité, matrix source×gravité
3. **Anomalies Détaillées** — Filtres avancés + table complète des anomalies
4. **Performance Financière** — CA par entité, business line, projet, tendances
5. **Audit Technique** — Tables brutes, statistiques, métriques d'audit
